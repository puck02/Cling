# Chapter 03: Functions

本章学习C语言的函数定义和使用。

## 📚 阅读材料

学习本章之前，建议先阅读：

- [C 函数](https://www.w3ccoo.com/c/c_functions.html) - 函数定义和调用
- [C 函数参数](https://www.w3ccoo.com/c/c_functions_parameters.html) - 参数传递详解
- [C 函数递归](https://www.w3ccoo.com/c/c_functions_recursion.html) - 递归函数

## 练习列表

1. **functions1.c** - 基础函数 ⭐
   - 学习函数定义和调用
   - 实现add函数

2. **functions2.c** - 多参数函数 ⭐⭐
   - 实现subtract, multiply, divide函数
   - 练习多个函数定义

3. **functions3.c** - 函数原型 ⭐⭐
   - 学习函数声明和定义分离
   - 理解函数原型的作用

4. **functions4.c** - return语句 ⭐⭐
   - 学习多返回路径
   - 实现max和min函数

5. **functions5.c** - 返回类型 ⭐⭐
   - 学习不同类型的返回值
   - circle_area返回double
   - is_positive返回int

6. **functions6.c** - 变量作用域 ⭐⭐⭐
   - 理解全局变量和局部变量
   - 学习函数访问全局变量

7. **functions7.c** - 递归函数进阶 ⭐⭐⭐⭐
   - 实现fibonacci和power递归函数
   - 掌握复杂递归模式

8. **functions8.c** - 递归函数入门 ⭐⭐⭐
   - 学习递归的基本概念
   - 实现简单递归（求和、阶乘、计数）

## 学习目标

- 掌握函数定义、声明和调用
- 理解参数传递和返回值
- 掌握函数原型的作用
- 理解变量作用域
- 学会使用递归解决问题
